#  Porównanie implementacji algorytmów skrótu (MD5, SHA1, SHA2) w C++, Python i C#

##  Opis projektu

Celem projektu jest implementacja i porównanie działania popularnych algorytmów skrótu:
- **MD5**
- **SHA-1**
- **SHA-256 (SHA-2)**

Każdy z algorytmów został zaimplementowany w trzech językach programowania:
- **C++**
- **Python**
- **C#**

Dodatkowy moduł porównujący umożliwia zestawienie wyników (hashy) oraz czasu wykonania poszczególnych implementacji.

---

##  C++ 
- **Compiler:** `g++` or `clang` (supporting C++11 or newer)
## Python
- **Python 3.10+**
## C#
- **.NET SDK 8.0 or newer**

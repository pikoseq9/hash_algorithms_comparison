#  Porównanie implementacji algorytmów skrótu (MD5, SHA1, SHA2) w C++, Python i C#

##  Opis projektu

Celem projektu jest implementacja i porównanie działania popularnych algorytmów skrótu:
- **MD5**
- **SHA-1**
- **SHA-256 (SHA-2)**

Każdy z algorytmów został zaimplementowany w trzech językach programowania:
- **C++**
- **Python**
- **C#**

##Instukcja

# Backend

1. Przejdź do katalogu backend : cd /backend
2. npm install
3. node index.js lub npm start

# Frontend

1. Przejdź do katalogu frontend : cd /frontend
2. npm install
3. npm run dev
